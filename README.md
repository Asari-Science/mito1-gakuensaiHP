# 第78回水戸一高・附属中学校学苑祭「天翔る」

第78回水戸一高・附属中学校学苑祭の公式サイトのソースコードです。

## プロジェクト構成

```
.
├── index.php              # トップページ
├── comingsoon.php         # Coming Soonページ（準備中ページ）
├── header.php             # 共通ヘッダー（ナビゲーション）
├── footer.php             # 共通フッター
├── menu.js                # ナビゲーションメニュー制御 + スクロールアニメーション
├── loading.js             # ローディング画面制御（sessionStorage対応）

├── header.css             # ヘッダー用スタイル
├── footer.css             # フッター用スタイル
├── comingsoon.css         # Coming Soonページ用スタイル

│   ├── enjitsu78th.webp   # 学苑祭ロゴ
```

## 実行方法

PHPがインストールされている環境であれば、以下のコマンドでローカルサーバーを起動して閲覧できます。

```bash
php -S localhost:8000
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

## 技術スタック

- **PHP**: 共通パーツ（ヘッダー・フッター）のインクルード
- **CSS3**: カスタムプロパティ、backdrop-filter、CSS Grid/Flexbox、アニメーション
- **JavaScript (Vanilla)**: メニュー制御、スクロールアニメーション、ローディング画面
- **Google Fonts**: Zen Old Mincho / Shippori Mincho / Noto Sans JP

## 使用しているWebフォント

| フォント名 | 用途 |
|---|---|
| Zen Old Mincho | タイトル、見出し、バッジ（重厚感） |
| Shippori Mincho | 本文テキスト、日程情報（上品さ） |
| Noto Sans JP | ラベル、サブテキスト、補助情報（モダンさ） |

## 主な機能

- **ローディング画面**: `NowLoading.gif`を使用。sessionStorageにより同一タブでの再読み込み時はスキップ、タブを閉じて再アクセス時には再表示
- **スクロールアニメーション**: Intersection Observer APIによるセクションのフェードイン
- **レスポンシブデザイン**: モバイル (〜400px) / タブレット (768px〜1023px) / PC (1024px〜) の3段階対応
- **ハンバーガーメニュー**: backdrop-filter対応のフルスクリーンオーバーレイメニュー
- **カウントダウンタイマー**: 開催日（2025年6月20日）までの残り日数をヘッダーに表示
- **Coming Soonページ**: 準備中のページ用テンプレート

## 開発ルール

- **コメントアウトは変更しない**: 他の開発者のコメントアウトしたコードは勝手にいじらないこと
- **共通パーツ編集時は全ページの影響を確認**: `header.php`、`footer.php`は全ページで読み込まれるため注意
- **フォント統一**: 新しいページを追加する場合は上記3種のGoogle Fontsを使用すること
- **カラーパレット**: `#f6dc9f`（背景）、`#dab584`（セクション背景）、`#1a1207`（テキスト）、`#8b6914`（アクセント）を基本とする
- **レスポンシブ対応必須**: 新しいセクション・ページは必ず3段階のブレークポイントに対応すること
