# 第78回水戸一高・附属中学校学苑祭「天翔る」

第78回水戸一高・附属中学校学苑祭の公式サイトのソースコードです。

## プロジェクト構成

- `index.php`: トップページ。
- `header.php`: 共通ヘッダー部分。
- `menu.js`: ナビゲーションメニューの制御用JavaScript。
- `reset.css`: CSSリセット。
- `header.css`: ヘッダー用スタイル。
- `toppage.css`: トップページ用スタイル。
- `materials/`: 画像や背景などの素材ディレクトリ。

## 実行方法

PHPがインストールされている環境であれば、以下のコマンドでローカルサーバーを起動して閲覧できます。

```bash
php -S localhost:8000
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

## 開発情報

- PHPを使用して共通部分（ヘッダーなど）のパーツ化を行っています。
- レスポンシブデザインに対応しています。
